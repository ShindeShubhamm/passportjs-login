# passportjs-demo

### This is a simple passport js demo.

1. Node, express
1. PassportJS
1. Handlebars (Templating Engine)

### Usage

Create `.env` file in root of your project.
Contents:

```env
PORT=3000
SESSION_SECRET=YourSessionSecret
```

Install dependencies:

```bash
npm install
```

Run demo

```bash
npm run dev # Runs development server (:3000)
```
